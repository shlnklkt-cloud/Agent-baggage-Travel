## How to Login

1. Go to the login page
2. Enter your **Passport/NRIC/FIN Number** (e.g., `CSGHY623JK`)
3. Enter your **Policy Number** (e.g., `INC-TRV-2024-79045`)
4. Click **Login**

## Features Available After Login

- File baggage delay claims
- File baggage loss/damage claims
- Lost travel documents claims
- Medical emergency claims
- Trip postponement claims
- Trip cancellation claims
- Trip shortening claims
- Trip disruption claims
- View travel itinerary
- Track claim status

---

**Note:** All users have active travel insurance policies and can access all claim features.
